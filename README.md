# Brain Tumor Frontend


![Exempt Development 2008 Frontend](images/brain-tumor.jpg)

This repository presents a brain tumor detection system built using Convolutional Neural Networks (CNNs) and ZenML, designed to provide an end-to-end, well-structured machine learning workflow. The project focuses on classifying MRI brain images to identify the presence and type of tumors, combining deep learning techniques with ZenMLâ€™s pipeline-based MLOps framework to ensure reproducibility, scalability, and clear experiment tracking. By organizing data preprocessing, model training, evaluation, and deployment into modular steps, the repository makes it easier to develop, validate, and maintain an accurate brain tumor detection solution suitable for research and real-world applications.
